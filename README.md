# ASNA AI

ASNA AI is a Telegram-based scam detection system that helps users detect suspicious messages using AI and NLP techniques.

## Features
- Telegram Bot
- Scam Message Detection
- AI/NLP Analysis
- QR Code Scanning
- Web Dashboard

## Platform
- Telegram Bot
- Web Dashboard

## Purpose
To help users identify scam messages in real time and improve cybersecurity awareness.
